# 熵灵 (Entropy Sprite)

> ⚠️ 由于本人没钱买显卡，没法装本地大模型。如果各位有钱可以装在本地大模型，就不用烧 Token 了，能实现真正的本地闭环且完全免费。本次测试所使用的模型为 GLM-4.7-Flash，是智谱 AI 的永久免费模型。

> 一个由**真随机**驱动的实验性意识模拟系统。它不是普通的聊天程序，而是一个拥有"内核冲动"与"外部语言"的双层心智模型。

## 关于我

我是一名六年级学生，会 C++。因为对意识和随机性感到好奇，我从十岁开始构思这个项目，十二岁终于动手实现了它。

## 我为什么这样设计

### 人类思考的秘密

我发现，人类思考的过程并不是直接"想"出答案。我们的大脑会先随机冒出很多乱七八糟的念头，然后自己给自己打分，把最合适的那个挑出来。这个过程太快了，所以我们根本察觉不到。

我就想：能不能用代码模拟这个过程？

### 1. 真随机驱动 — 模拟大脑噪声

我用操作系统硬件熵来决定要不要说话、说几句话、每句话多长。然后通过随机 ASCII 码生成最原始的念头。它无时无刻都在随机——就跟大脑里的神经噪声一样，是想法诞生的土壤。

### 2. 自我审查 — 语义级批量审查官

生成的乱码会交给审核官。审核官调用大模型，根据视觉模型看到的场景和情绪状态，从语义层面批量判断所有念头是否有意义。审核官会根据视觉关键词从记忆库中检索相关记忆，审查结果会被缓存，相同场景不重复调用 API。

反驳冲动：当念头连续被否定时，系统会累积反驳冲动。审核官评分时会对边缘分数更宽容，发言概率也会临时提高。

### 3. 场景分类 — 云端大模型 + 本地 jieba 兜底

视觉模型优先调用云端大模型进行分词、词性标注和主要描述对象识别。分类结果会被缓存。如果云端调用失败，自动降级为本地 jieba 分词。

### 4. 情绪模型 — 模拟内在状态

三维情绪系统：精力、好奇心、愉悦。情绪随时间衰减，被事件影响。审核官打出低分时好奇心自然增长。情绪直接影响发言冲动。

### 5. 记忆系统 — 遗忘与巩固

三层记忆，模仿人脑：短期记忆一天删一次，长期记忆一年删一次，永久记忆永远不删。一天内被连续提及 N 次就短期升长期，一年内被提 N 次就长期升永久。所有记忆用 SQLite 存在硬盘上，关机重启不丢失。

联网学习：思考者遇到不认识的概念时自动联网搜索，用大模型总结核心特征后存入记忆库。同一关键词一天内不重复搜索。

### 6. 生成策略 — 四层优先级

1. hint 引导：审核官反馈的最相关记忆概念作为方向
2. 词素拼图：从记忆碎片中随机拼接新词
3. 整词插入：从记忆库取完整片段
4. 兜底随机：纯随机字母，只在记忆库完全为空时启用

### 7. 沉默时的内心闪过

当内核决定不说话时，仍会生成简短念头交给审核官判断。有意义就附在回答末尾，无意义则留下低调的省略号。

### 8. 解答者 — 类比推理引擎

遇到问题时从记忆库中检索相关概念，构建类比链，然后基于类比推理给出答案。思考者联网学习时会顺便解答问题，一次 API 调用完成两件事。

### 9. 规划者 — 意图到动作的拆解

当文本中包含物理动作意图时，自动拆解为具体的动作步骤。先用本地规则过滤，没有动作词直接跳过。

### 10. 虚拟环境 — 2D 网格世界

在 5×5 网格中模拟世界状态。AI 可以用自然语言动作与世界交互：向北移动、抓取苹果、放下石头、观察周围。动作执行结果会写进场景描述，AI 能感知世界变化。

## 模块说明

* randomness.py — 从操作系统熵池提取真随机数
* memory.py — 三层记忆系统，SQLite 持久化
* emotion.py — 情绪模型，随时间衰减，状态保存
* vision.py — 视觉模型，支持三种输入模式
* classifier.py — 云端语义分类器，带缓存
* thinker.py — 思考者，四层生成策略 + 联网学习
* reviewer.py — 语义审查官，批量审查
* language_model.py — 大模型外壳 + 本地兜底
* solver.py — 解答者，类比推理
* planner.py — 规划者，意图拆解
* virtual_env.py — 虚拟环境，自然语言交互
* test_virtual.py — 自动测试脚本
* main.py — 主系统，整合所有模块

## 使用指南

### 1. 下载项目

**方法一：用 Git 克隆（推荐）**

打开终端（Windows 用 PowerShell 或 CMD），输入：

**text**

```
git clone https://github.com/DavidDengll/The-new-living-things.git
```

克隆完成后，进入项目文件夹：

**text**

```
cd The-new-living-things
```

**方法二：直接下载 ZIP**

如果你不会用 Git，打开 GitHub 仓库页面：

[https://github.com/DavidDengll/The-new-living-things](https://github.com/DavidDengll/The-new-living-things)

点击页面右上角的绿色 "Code" 按钮，选择 "Download ZIP"。下载后解压到任意文件夹，然后打开终端进入解压后的文件夹。

### 2. 安装依赖

在项目文件夹里打开终端，输入：

**text**

```
pip install -r requirements.txt
```

等待安装完成。

### 3. 获取 API Key

1. 打开浏览器，访问智谱AI开放平台：[https://open.bigmodel.cn/](https://open.bigmodel.cn/)
2. 注册账号并完成实名认证
3. 进入控制台，点击"API 密钥"
4. 创建一个新的 API Key，复制完整 Key（以 `sk-` 开头）
5. 用记事本打开项目里的 `config.py` 文件
6. 找到这一行：`ZHIPU_API_KEY = "你的真实API-KEY"`
7. 把引号里的内容替换成你复制的 Key，保存文件

### 4. 运行

**键盘交互模式：**

**text**

```
python main.py
```

运行后每轮提示你输入场景描述。直接回车使用默认场景，输入 `Ctrl+C` 退出。

**虚拟环境自动测试：**

**text**

```
python test_virtual.py
```

在 5x5 的网格中自动运行 100 轮测试。AI 会自动移动、抓取物品、放下物品。终端会显示网格状态和 AI 的思考过程。同样按 `Ctrl+C` 退出。

所有可调参数集中在 config.py：发言冲动、念头通过线、重试次数、记忆升级条件等。视觉模式可在 keyboard、simulate、real 之间切换。

### 关于模型费用

本次测试使用 GLM-4.7-Flash，是智谱 AI 的永久免费模型。有钱买显卡的话可以把 API 换成 Ollama 等本地模型，实现完全免费的本地闭环。
